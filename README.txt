Корпоративный сайт написанный на ларавель.
1. Главная страницы с выведеными сеткой портфолио, навигационное меня сделано с помощью laravel-menu
2. Вся информация и картинки грузятся из БД. Картинки сохранены в БД в json формате,чтобы были пути сразу на три изображения нужного поля разного размера
3. сделан сайдбар справа с последними записями в блоге
4. в блоге выводятся все новости и по категориям,справа сайдбар с последними порфтолио и ниже блок с последними комментариями
5. Использована русская локализация для заголовков блоков
6. в комментариях указывается имя того кто писал,название новости к которой комментарий был написан,и ссылка на новость с комментарием
8. Если пользователь не заругистрирован, то вместо аватарки грузится стандартная заглушка
9. на странице просмотра одной новости выводится большое изображение новости,снизу блок комментариев в древовидном виде,чтобы было ясно кто на какой комментарий отвечал
10.сделана страница со списком портфолио, так же динамически формируется каждая страница просмотра отдельного портфолио
11. сделана проверка пути маршрута на удовлетворение регулярному выражению для безопасности
12. изменен метод выдачи ошибки 404, теперь выдается не стандартная страница ошибки а кастомная
13. переопределен метод showLoginForm в LoginController чтобы отображаемая страница входа была кастомная и находилась в нужной нам папке,и передали параметр заголовка страницы.
14. делаем админпанель, выставляем условие что только пользователь с правами может туда зайти
15. сделана страница на админ панели для управления статьями(редактирование,добавление,удаление)
16. добавлен метод автоматической генерации псевдонима статьи,который должен быть уникальным,в противном случае пользователь получает сообщение об ошибке и может исправить псевдоним
17. ресайз загружаемых изображений в статьи и запись в БД с помощью json чтобы были доступна все три размера изображения
18.добавлен раздел в админпанели для редактирование прав ролей пользователей
19.добавлен раздел для создания и редактирования пунктов меню на сайте,реализован акордион jquery для выбора типа меню
